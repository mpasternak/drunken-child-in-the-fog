============
Installation
============


Stable release
--------------

To install Drunken Child In The Fog, run this command in your terminal:

.. code-block:: console

    $ pip install drunken_child_in_the_fog

This is the preferred method to install Drunken Child In The Fog, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Drunken Child In The Fog can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mpasternak/drunken_child_in_the_fog

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/mpasternak/drunken_child_in_the_fog/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mpasternak/drunken_child_in_the_fog
.. _tarball: https://github.com/mpasternak/drunken_child_in_the_fog/tarball/master
